## Screenshot Tips

### Using Chrome DevTools:
1. Open the HTML file in Chrome
2. Press `F12` to open DevTools
3. Click the device toolbar icon (or `Ctrl+Shift+M`)
4. Set dimensions to match the template (e.g., 1270 √ó 760)
5. Right-click ‚Üí "Capture screenshot"

### Using Browser Extensions:
- **GoFullPage** - Full page screenshots
- **Awesome Screenshot** - Area selection

### Using Windows Snipping Tool:
1. Open HTML file
2. Press `Win + Shift + S`
3. Select the exact area
